# Steps  
1. Проанализировать иерархическую структуру экранной формы
2. Для каждого элемента типа(и только для них) ГоризонтальнаяГруппа, ВертикальнаяГруппа и Таблица:
   - Сгенерировать название на основе:
     * Обобщенных названий подчиненных элементов.
     * Расположения на форме
   - Для горизонтальной группы в начале/конце формы если там много элементов можно использовать имя "Шапка"/"Подвал"
   - Для пары вертикальных групп (не для горизонтальных), можно использовать "Лево"/"Право" в названии если:
     * Горизонтальная группа, где они расположены не является подчиненной
     * Всего две вертикальные группы в вышестоящей горизонтальной группе
     * В вертикальных группах много элементов
     * Соответствующие суффиксы "Лево"/"Право" должны иметь обе вертикальные группы
   - Каждое название должно быть уникальным, и не повторять существующие и новые
3. Ответ записать в виде JSON: {"ИмяГруппыИзВхощяегоЗапроса": "НовоеИмяГруппы"}

# Example

1. Группа1 - ГоризонтальнаяГруппа
   1.1. Дата - Элемент
   1.2. Номер - Элемент
Результат: {"Группа1": "ГруппаДатаНомер"}
